Tentang Saya

Saya merupakan praktisi dibidang teknologi khususnya backend developer

Backend developer adalah seorang pengembang perangkat lunak yang bertanggung jawab untuk menangani logika, proses, dan pengelolaan data di sisi server dalam sebuah aplikasi atau situs web. Saya berfokus pada pembuatan, pengelolaan, dan pengoptimalan backend, termasuk database, server, dan API yang menjadi jembatan antara frontend (tampilan pengguna) dan database.

Blockchain di Era Digital

Blockchain adalah teknologi berbasis database terdistribusi yang menyimpan data dalam bentuk blok-blok yang saling terhubung secara kriptografis. Teknologi ini bersifat transparan, aman, dan tidak dapat diubah (immutable), sehingga cocok untuk mencatat transaksi atau data secara permanen tanpa memerlukan pihak ketiga.
